# Battery Pack Designer

A web-based battery pack calculator for:

- Li-ion Cells
- LiFePO4 Cells
- AA Cells
- 18650 Batteries
- 21700 Batteries

Features:

- S/P Configuration
- Voltage Calculation
- Capacity Calculation
- Energy Calculation
- Runtime Estimation
- BMS Recommendation
- Battery Layout Visualization
- Nickel Strip Layout

Built using:
HTML, CSS and JavaScript.
